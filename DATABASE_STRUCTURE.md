# 2.X Структура хранения данных проекта

Все данные проекта хранятся в СУБД `PostgreSQL` и `MongoDB`.

В СУБД `PostgreSQL` хранятся таблицы пользователей, профилей, обращений, журнала действий, чатов, связей пользователей, прав доступа и служебных кодов.

В СУБД `MongoDB` хранятся коллекции файлового сервиса: файлы, комнаты, пользовательские хранилища, элементы хранилищ и сессии загрузки.
